### utools-fund

- 自选基金助手，数据来源于天天基金
- 支持新增删除自选基金，可填入持有份额，实时计算当天收益

#### 源码

> [https://github.com/adams549659584/utools-fund](https://github.com/adams549659584/utools-fund)
